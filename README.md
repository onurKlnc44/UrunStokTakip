# Ürün Stok Takip

ASP.NET Core MVC ile geliştirilmiş, ürün ekleme, güncelleme, silme ve stok takibi yapan bir web uygulaması.

---

## Özellikler

- Ürün ekleme, güncelleme ve silme
- Stok miktarına göre uyarı (badge)
- Responsive tasarım, mobil uyumlu
- Bootstrap 5 ile modern UI
- Veritabanı (Entity Framework Core ile MSSQL)
- Başarı mesajları (ürün eklendi, güncellendi, silindi)
